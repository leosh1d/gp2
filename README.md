## Тема

Факторы роста зарплат и выгодные комбинации навыков на рынке вакансий IT-профессий

## Бизнес-задача

Определить, какие факторы (навыки, их комбинации, локация, профессия и дефицит кадров) статистически связаны с более высоким уровнем заработной платы, чтобы сформировать рекомендации по разработке образовательных программ в IT-сфере.

## Заказчик

онлайн-платформа образования / edtech по ИТ-профессиям

например:
- Skillbox
- Нетология
- Яндекс Практикум

## Ценность для заказчика
- запуск курсов по действительно востребованным навыкам
- рекомендации ученикам и возможные партнерства с компаниями, которые платят более конкурентную ЗП
- запуск новых направлений (например, IT + финансы)


## Сбор данных
Для получения данных для анализа рынка вакансий мы использовали следующие источники: Superjob и hh.ru.
### API superjob
Чтобы получить данные с Superjob мы зарегистрировали наш проект на https://api.superjob.ru. Этой документацией мы и пользовались при дальнейшем сборе данных. 

После регистрации мы получили секрет для получения токена SUPERJOB_SECRET. Нужно скопировать example.env в .env и указать полученнный SUPERJOB_SECRET. Мы применили 5 разных методов АПИ (get catalogues, get vacancies, get towns, get regions, get countries) и получили датасет с различной информацией о вакансиях с Superjob.
### Скрепинг и парсинг hh.ru
Основную часть итогового датасета составляют вакансии с [hh.ru](https://hh.ru). С помощью библиотеки Selenium мы получили данные по вакансиям в сфере ИТ.

**Пояснение признака vac_description:**
1. Изначально текстовое описание вакансии - полное описание из Superjob и краткое из hh (тк скрепингом было долго залезать в каждую вакансию и получать полное описание, то мы спарсили краткое в режиме расширенного описания вакансий)

2. С помощью [внешнего датасета](https://github.com/iconicompany/iskillmatching/tree/main?tab=readme-ov-file#4-нормализация-через-векторные-представления) с навыками, мы извлекли из описаний вакансий по шаблону эти навыки и нормализовали (привели к единому виду).
