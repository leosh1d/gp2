# Тема

Факторы роста зарплат и выгодные комбинации навыков на рынке вакансий

### Бизнес-задача

Определить, какие факторы (навыки, их комбинации, локация, профессия и дефицит кадров) статистически связаны с более высоким уровнем заработной платы, чтобы сформировать рекомендации по разработке образовательных программ.

## Заказчик

онлайн-платформа образования / edtech
например:
 • Skillbox
 • Нетология
 • Яндекс Практикум

### Ценность для заказчика
 • запуск курсов с высокой ROI (учат тому, что реально повышает зарплату)
 • упаковка курсов: “освой X + Y → +30% к доходу”
 • выбор новых направлений (например, IT + финансы)
